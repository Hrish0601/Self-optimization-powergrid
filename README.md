# Self-optimization-powergrid
An intelligent smart-grid management system that combines Machine Learning, Reinforcement Learning, and Cloud Computing to improve electricity theft detection, load balancing, and energy optimization in modern power grids.  
An intelligent smart-grid management system that combines Machine Learning, Reinforcement Learning, and Cloud Computing to improve electricity theft detection, load balancing, and energy optimization in modern power grids.

Project Overview

This project focuses on building a self-optimizing electric grid capable of:

Detecting electricity theft using ML models
Balancing energy demand dynamically using Reinforcement Learning (PPO)
Generating realistic synthetic grid data using Gaussian Copula models
Deploying scalable cloud-based inference pipelines using Google Cloud services

The system integrates fraud detection and load management into a single intelligent framework for real-time smart-grid operations.

 Key Features
 Electricity Theft Detection
Detects abnormal consumption patterns from smart meter data
Uses ML classification models with explainable predictions
Handles class imbalance using synthetic data generation
Reinforcement Learning for Load Balancing
Uses Proximal Policy Optimization (PPO) for dynamic energy distribution
Optimizes load scheduling and energy utilization
Reduces overload and improves grid stability

Synthetic Data Generation
Gaussian Copula-based synthetic data modeling
Preserves real-world feature correlations
Improves training robustness and privacy
Cloud-Native Architecture
Google BigQuery for storage and analytics
Pub/Sub for real-time streaming
Vertex AI for scalable model deployment
Flask APIs for integration
